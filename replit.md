## Purpose
- Educational Python exercises for learning programming fundamentals
- Hands-on practice with Python syntax and concepts
- Organized progression through different programming topics

## Current State
- **Language**: Python 3.11
- **Type**: Console/CLI application with interactive menu
- **Dependencies**: None (uses only Python standard library)
- **Entry Point**: `main.py` provides an interactive menu to select and run different exercises

## How to Use
Run `main.py` to launch the interactive menu system. From there, you can:
1. Select a class or activity category
2. Choose a specific script to run
3. View the output and return to the menu
